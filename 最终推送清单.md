## ⚠️ 重要提醒

### 关于主题配置
主题目录 `themes/hexo-theme-matery` 在 `.gitignore` 中被排除，但我们使用了 `git add -f` 强制添加了 `_config.yml` 配置文件。

**建议**：
- 如果你自定义了主题，考虑将整个主题从 `.gitignore` 中移除
- 或者使用 git submodule 管理主题

### 关于 package.json 和 package-lock.json
这两个文件有未提交的修改，可能是 npm 自动更新的。

**检查并提交**：
```bash
# 查看修改内容
git diff package.json
git diff package-lock.json

# 如果是正常的版本更新，提交它们
git add package.json package-lock.json
git commit -m "更新依赖版本信息"
```
