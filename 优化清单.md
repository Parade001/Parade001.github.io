## ⚠️ 需要手动处理的优化

### 1. 依赖包更新（高优先级）
**当前版本严重过时：**
- Hexo: 5.4.2 → 8.1.2（落后 3 个大版本）
- Gulp: 4.0.2 → 5.0.1
- 多个 hexo 插件需要更新

**更新步骤：**
```bash
# 方案 1: 保守更新（推荐）
npm update

# 方案 2: 大版本升级（需要测试）
npm install hexo@latest --save
npm install gulp@latest --save

# 更新所有插件
npm update hexo-deployer-git hexo-generator-sitemap hexo-renderer-stylus
```

⚠️ **风险提示**: Hexo 5 → 8 是大版本升级，可能存在破坏性变更，建议先备份后测试。

### 2. 移除冗余 Markdown 渲染器（中优先级）
**问题**: 同时安装了 `hexo-renderer-markdown-it` 和 `hexo-renderer-marked`

**解决方案：**
```bash
# 推荐保留 markdown-it（功能更强大）
npm uninstall hexo-renderer-marked

# 或者保留 marked（更轻量）
npm uninstall hexo-renderer-markdown-it
```

### 3. 安全漏洞修复（高优先级）
**严重问题**: Gitalk OAuth clientSecret 暴露在配置文件中

**位置**: `themes/hexo-theme-matery/_config.yml:342-343`
```yaml
oauth:
  clientId: 08cb8efd61d2c27103a2
  clientSecret: 5a2cb78fe45a079b772fc565e7f4b758d09bc07f  # ⚠️ 已泄露
```

**解决方案：**
1. 立即前往 GitHub Settings → Developer settings → OAuth Apps
2. 重新生成 Client Secret
3. 考虑切换到更安全的评论系统（如 Utterances）

### 4. 主题管理优化（中优先级）
**问题**: 主题目录在 `.gitignore` 中被排除

**解决方案：**
```bash
# 方案 1: 使用 git submodule
git submodule add https://github.com/blinkfox/hexo-theme-matery themes/hexo-theme-matery

# 方案 2: 移除 .gitignore 规则并提交主题
# 编辑 .gitignore，删除 themes/hexo-theme-matery 这行
```

### 5. 图片优化（中优先级）
**问题**: public 目录 107MB，可能包含未优化的图片

**解决方案：**
```bash
# 清理并重新构建（会自动压缩图片）
npm run clean
npm run build:prod
```
